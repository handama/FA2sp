# Actions参数表

触发行为(Actions)存储在map文件中的格式为
```ini
[Actions]
触发id=总数,ID1,P2,P3,P4,P5,P6,P7,P8,ID2,...
```

`Actions`参数指`"ID1,P2,P3,P4,P5,P6,P7,P8"`这样具有8个参数的字符串，其中参数P3可以为字符串，P8可以为路径点字母格式, 

例如: `01000004=2,41,0,8,1,0,0,0,Z,50,0,0,0,0,0,0,A`, `2`代表行为总数, `41,0,8,1,0,0,0,Z`代表第一个行为, `50,0,0,0,0,0,0,A`代表第二个行为。     

[section] 代表外部参数, 需在 `external_params.md` 中查找   

P8在不同Action中要求完全不同的编码格式，触发参数 P8的 <路径点字母格式> 需要调用Lua脚本文档.md的函数`waypoint_to_string(wp)` 和 `string_to_waypoint(str)` 来与10进制数字相互转化。
标注 (无效) 的触发行为id请避免使用它，如果使用必须要给用户必要的提醒。

当考虑使用拓展平台（包括Ares和Phobos等）的Action时，要给以用户足够的提示，优先使用原版的Action来写。
