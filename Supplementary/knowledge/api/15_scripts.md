# 十四、动作脚本 (`Script`)

## `Script` 类

### `Script:new(id)` （构造函数）
- **说明**：构造动作脚本对象。
- **参数**：`id` (`string`) — 脚本 ID（可用 `get_free_id(4)` 获取）。
- **返回** (`Script`)：脚本对象。

**成员**：

| 属性 | 类型 | 读写 | 说明 |
|------|------|------|------|
| `id` | `string` | 只读 | 脚本 ID |
| `name` | `string` | 读/写 | 脚本名称（第 0 个参数） |
| `actions` | `ScriptActionCollection` | 读/写 | 动作列表（第 1 个参数起） |

**方法**：

### `apply()`
- **说明**：将脚本写入地图 INI 并刷新。
- **返回**：无。

### `delete()`
- **说明**：从地图 INI 中删除该脚本。
- **返回**：无。

## 全局函数

### `get_script(id)`
- **说明**：通过 ID 获取动作脚本对象。未找到时返回 `name` 为空的无效对象。
- **参数**：`id` (`string`) — 脚本 ID。
- **返回** (`Script`)：脚本对象。

### `get_scripts()`
- **说明**：获取所有脚本的 ID 数组。
- **返回** (`table<string>`)：ID 数组。
