# 红色警戒2术语中英对照表（Red Alert 2 Terminology Chinese–English Glossary）

| 中文 | English | 别称 | 说明 |
|------|-------|------|------|
| 地图类型 | Theater | 气候 | 地图地表的基本类型，包括温和、雪地、城市、新城市、月球、沙漠六种 |
| 温和 | TEMPERATE | 草地、温带 | 地图类型之一 |
| 雪地 | SNOW |  | 地图类型之一。一般情况下，其他地图类型的建筑图像均一致，而雪地类型则需要特殊的覆雪图像 |
| 城市 | URBAN | 二班、旧城市 | 地图类型之一 |
| 新城市 | NEWURBAN | 牛二班、新城 | 地图类型之一 |
| 月球 | LUNAR | 太空 | 地图类型之一 |
| 沙漠 | DESERT |  | 地图类型之一 |
| 游戏对象 | Object | 对象 | 所有游戏内的实体对象 |
| 覆盖物 | Overlay | 覆盖图、覆盖 | 一种放置在单元格上的结构，常见的有金矿、围墙、桥梁等。一般不被称为游戏对象 |
| 覆盖物数据 | OverlayData | 覆盖图数据 | 规定了某种覆盖物的变体，比如破损的围墙 |
| 建筑 | Structure | 建筑物、building | 固定的、可以占据多个单元格的游戏对象 |
| 步兵 | Infantry |  | 可动的游戏对象，一个单元格内最多可以有三个步兵 |
| 车辆 | Unit | vehicle、坦克、战车 | 可动的游戏对象（一般而言，直升机或空艇都属于车辆） |
| 飞行器 | Aircraft | 飞机 | 可动的游戏对象 |
| 科技类型 | Techno | | 包括建筑、步兵、车辆、飞行器，游戏内玩家只能操作科技类型 |
| 单位 | Footclass |  | 所有可动的科技类型，包括步兵、车辆、飞行器 |
| 地形对象 | TerrainType | 树木、tree、terrain | 不可动的游戏对象，一般包括树木、矿井、路灯等。虽然其中文中有“地形”（英文中有“Terrain”），但它并不是地形 |
| 单元标记 | Celltag |  | 放置于单元格上，表明这个单元格关联到了某一特定标签 |
| 基地节点 | base node | 节点 | 仅用于单人地图，用于规划AI所属方如何以及在哪里建造建筑 |
| 隧道 | Tube | tunnel | 放置于地图两个单元格上的特殊结构，可以让单位在两个单元格之间以地下潜行的方式移动 |
| 路径点 | Waypoint | 航点 | 放置于地图单元格上的标记，一般用于小队或触发，如在让小队成员移动到路径点，在路径点释放超级武器 |
