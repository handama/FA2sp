# 附录

## A. 消息框格式 (`format` 参数)

| format | 按钮 | 图标 |
|--------|------|------|
| 1 | 确定 | 无图标 |
| 2 | 是/否 | 疑问 |
| 3 | 是/否 | 警告 |
| 4 | 是/否/取消 | 疑问 |
| 5 | 是/否/取消 | 警告 |
| 6 | 确定/取消 | 警告 |
| 7 | 是/否 | 信息 |
| 8 | 确定 | 错误 |
| 9 | 是/否/取消 | 信息 |

用户按下按钮的返回值从 1 开始递增。
