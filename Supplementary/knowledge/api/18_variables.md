# 十七、变量

## `variable` 类

### `variable:new(name)` （构造函数）
- **说明**：构造变量对象。
- **参数**：`name` (`string`) — 变量名（显示名称和内部 Key 均为此值）。
- **返回** (`variable`)：变量对象。

**成员**：

| 属性 | 类型 | 读写 | 说明 |
|------|------|------|------|
| `name` | `string` | 读/写 | 变量名（唯一标识） |
| `state` | `number` | 读/写 | 当前状态值 |
| `default` | `number` | 读/写 | 默认状态值 |
| `desc` | `string` | 读/写 | 变量描述 |
| `timestamp` | `string` | 读/写 | 时间戳 |
| `comment` | `string` | 读/写 | 备注 |

**方法**：

### `apply()`
- **说明**：将变量写入地图 INI 并刷新。
- **返回**：无。

### `delete()`
- **说明**：从地图 INI 中删除该变量。
- **返回**：无。

## 全局函数

### `get_variable(name)`
- **说明**：通过名称获取变量对象。未找到时返回 `name` 为空的无效对象。
- **参数**：`name` (`string`) — 变量名。
- **返回** (`variable`)：变量对象。

### `get_variables()`
- **说明**：获取所有变量名称的数组。
- **返回** (`table<string>`)：变量名称数组。
