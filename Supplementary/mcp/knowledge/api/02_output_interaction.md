## 二、输出与交互

### `print(...)`
- **说明**：在输出窗口打印文本。可以接受任意数量和类型的参数，每个参数用空格分隔。
- **参数**：`...` — 可变参数，支持 `nil`, `string`, `number`, `boolean`。
- **返回**：无。
- **示例**：`print("Hello", 123, true)`

### `clear()`
- **说明**：清空输出窗口中的文本。
- **返回**：无。

### `avoid_time_out()`
- **说明**：向主程序分发一次消息，防止脚本执行时间过长导致界面卡死（“未响应”）。在长时间循环中应适时调用。
- **返回**：无。
- **使用示例**：
```lua
for i = 1, 1000 do
    -- 大量操作
    if i % 50 == 0 then
        avoid_time_out()
    end
end
```

### `sleep(ms)`
- **说明**：暂停脚本执行指定的毫秒数。
- **参数**：`ms` (`number`) — 毫秒数。
- **返回**：无。

### `end_script()`
- **说明**：立即终止当前 Lua 脚本的执行。
- **返回**：无。
